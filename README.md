### Installation
```bash
$ [sudo] pip install s3-create-user
```

```bash
$ [sudo] npm i -g s3-create-user
```

#### Pros
+   create S3 bucket user, policy and access key

#### Examples
```bash
$ s3-create-full-access-user bucket_name
$ s3-create-read-only-user bucket_name
```
